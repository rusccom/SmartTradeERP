# Tables UI

Здесь живут разные типы таблиц. Не смешивай сценарии.

- `list-table` — универсальная read-only таблица для списков сущностей: товары, клиенты, документы, отчеты.
- `document-editable-table` — будущая редактируемая таблица строк внутри карточки документа.

Если страница показывает уже готовый список записей с server-side pagination/search/filter/sort, используй `list-table`.
Если пользователь редактирует строки одного документа до сохранения, используй `document-editable-table`.
