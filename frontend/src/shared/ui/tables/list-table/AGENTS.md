# List Table

Эта папка содержит универсальную read-only таблицу списков.

Используй ее для страниц, где нужно показать список готовых сущностей:
- документы;
- товары;
- клиенты;
- справочники;
- будущие list-отчеты.

Границы компонента:
- таблица не редактирует ячейки;
- данные приходят извне через controlled props;
- сортировка, поиск, фильтры и пагинация работают server-side;
- API-состояние хранится в `shared/model/data-table`;
- бизнес-логика фич остается в preset файлах фичи.

Не добавляй сюда inline edit, document item rows, draft-calculation логику или сохранение формы документа.
