# Document Editable Table

Эта папка зарезервирована для редактируемой таблицы строк внутри карточки документа.

Используй ее только для сценария, где пользователь собирает или редактирует один документ:
- строки прихода;
- строки продажи;
- строки списания;
- строки перемещения;
- строки возврата.

Границы будущей реализации:
- строки редактируются локально как часть формы документа;
- компонент не должен подменять `list-table`;
- пагинация списка документов здесь не нужна;
- загрузку справочников и сохранение документа делает фича документа;
- компонент должен отдавать изменения наружу через controlled props.

Не используй эту папку для страниц списков `/dashboard/documents`, `/dashboard/products` или `/dashboard/customers`.
